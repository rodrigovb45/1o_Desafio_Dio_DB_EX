# Principais pontos da revisão Git e Github

**Git** - Sistema de versionamento distribuído

**Github** - Respoitório na nuvem

##Benefícios da utilização do Git/Github
1 - Controle de versão
2 - Armazenamento em nuvem
3 - Facilitador do trabalho em equipe
4 - Melhoria do seu código através do trabalho compartilhado em equipe

# Comandos básicos apresentados

**git clone*- para clonar o  repositório na sua máquina.
**git add.** - para atualizar as informações do arquivo para o git.
**git commit** - para "commitar" o código. Destacando aqui a importância do cometáriono  commmit.
**git push* - para enviarpara a nuvem.
